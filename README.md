## Installation

Install via composer as any other Magento extension from github:
```
composer require jajuma/module-hyva-pwa
```

## Using Hyvä Themes as PWA with JaJuMa "Hyvä PWA" module

Go to **JaJuMa -> Hyvä PWA -> Configuration**  
And complete the configurations for manifest and service worker.

* [See Manual for more details](https://www.jajuma.de/sites/default/files/ckfinder/userfiles/images/jajuma-develop/hyva-pwa-hyva-themes-magento/JaJuMa_Hyva_PWA_v001.pdf)
